> A curated list of awesome things related to Seed

[Seed](https://seed-rs.org/) is an open-source Rust framework for creating fast and reliable web apps running in WebAssembly.

Contributions welcome. Add links through pull requests or create an issue to start a discussion.

## Quickstarts

- [Default quickstart](https://github.com/seed-rs/seed-quickstart) - Contains only Rust libraries. :star:13
- [Quickstart with Webpack](https://github.com/seed-rs/seed-quickstart-webpack) - Main features: Auto-reload, Prerendering, Minification, [TailwindCSS](https://tailwindcss.com/) and Typescript integration. :star:28

## Examples

- [RealWorld example](https://github.com/seed-rs/seed-rs-realworld) - "The mother of all demo apps" — Exemplary fullstack [Medium.com](https://medium.com/) clone. :star:21
- [Official examples](https://github.com/seed-rs/seed/tree/master/examples) - Smaller examples included in official repo. :star:737
- [ERWA mytodo](https://github.com/seed-rs/erwa_mytodo) - Rust full-stack example. Diesel, Rocket, Seed.

## Projects Using Seed

- [AdEx Explorer](https://github.com/adexnetwork/adex-explorer) - Shows curated information about the payment channel network of the AdEx advertising protocol. :star:3
- [Kavik.cz](https://github.com/MartinKavik/kavik.cz) - Open-source personal website. :star:2
- [benxu.dev/blog](https://github.com/AlterionX/benxu-dev) - A relatively simple open source personal blog. Built on [`Seed`](https://seed-rs.org/), [`maud`](https://maud.lambda.xyz), [`Rocket`](https://rocket.rs), and [`Diesel`](https://diesel.rs).
- [seed-rs.org](https://github.com/seed-rs/seed-rs.org) - Seed's official website. :star:1
